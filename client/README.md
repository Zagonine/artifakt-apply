# Client installation

## Requirement
- NodeJS LTS version 10.15.x

## Setup project
- Install `node_modules` with command `$ npm install`

# Run project

## Run project in dev mode
- `$ npm run start`

## Build project
- `$ npm run build`

# Run tests
- `$ npm run test`

# Others commands

## Lint
- `$ npm run lint`
